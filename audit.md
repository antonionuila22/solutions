Descripción del problema
Los códigos de estado HTTP 4XX indican que no se puede
acceder a la página o recurso solicitado. Las URLs 4XX ofrecen
una mala experiencia de usuario para los visitantes del sitio web,
ya que estos no pueden acceder a la página o archivo a través
del enlace en el que hicieron clic. Además, los enlaces internos a
URLs 4XX pueden reducir tu presupuesto de rastreo. Las páginas

serán eliminadas del índice de los motores de búsqueda.

se enlaza a una URL 4XX. Elimina los enlaces rotos o
reemplázalos con enlaces apropiados hacia páginas accesibles.

mover páginas de tu sitio web para evitar errores 4XX. El código
de respuesta HTTP 429 (Demasiadas Solicitudes) puede indicar
que la frecuencia del rastreo especificada en los ajustes del
rastreo para tu proyecto es demasiado alta para que el servidor
la pueda soportar. Baja el límite en la configuración (Límites y
restricciones) y vuelve a escanear el proyecto.
